## نصب پلاگین

1. **فایل‌ها را آماده کنید**
2. **پوشه پلاگین بسازید:** `haika-multilevel-menu`
3. **آپلود کنید:** در `/wp-content/plugins/`
4. **فعال کنید:** از پنل ادمین > پلاگین‌ها

## راهنمای کامل استفاده

### مرحله 1: ساخت منو
1. پنل ادمین > **ظاهر > منوها**
2. کلیک **"ایجاد منوی جدید"**
3. نام منو (مثلاً "منوی اصلی")
4. اضافه کردن صفحات و لینک‌ها
5. **مهم:** در بخش "مکان‌های منو" گزینه **"منوی چندسطحی هایکا"** را انتخاب کنید
6. ذخیره منو

### مرحله 2: تنظیم آیکون و ظاهر
1. پنل ادمین > **منو هایکا**
2. **متن دکمه:** متنی که کنار آیکون نمایش داده می‌شود
3. **آیکون دکمه:**
   - کلیک **"انتخاب/تغییر آیکون"**
   - از کتابخانه رسانه یا آپلود جدید
   - انتخاب عکس مناسب
4. **اندازه آیکون:** 16 تا 50 پیکسل
5. **رنگ دکمه:** انتخاب رنگ پس‌زمینه
6. **ذخیره تنظیمات**

### مرحله 3: نمایش منو

#### در Elementor:
1. ویرایش صفحه با Elementor
2. اضافه کردن ویجت **"Shortcode"**
3. وارد کردن کد: `[haika_menu]`

#### در ویجت‌ها:
1. ظاهر > ویجت‌ها
2. ویجت **"متن"** یا **"HTML سفارشی"**
3. کد: `[haika_menu]`

#### در فایل‌های قالب:
```php
echo do_shortcode('[haika_menu]');
```

## شورت‌کدهای موجود

```php
// پایه
[haika_menu]

// تغییر متن
[haika_menu button_text="منوی اصلی"]

// تغییر آیکون
[haika_menu button_icon="https://example.com/icon.png"]

// تغییر اندازه آیکون
[haika_menu icon_size="24"]

// تغییر موقعیت منو (برای گوشه چپ)
[haika_menu position="left"]

// فقط آیکون (متن خالی)
[haika_menu button_text="" button_icon="https://example.com/icon.png"]

// ترکیبی
[haika_menu button_text="منو" icon_size="20" position="left"]
```

## نکات مهم برای آیکون

### بهترین فرمت‌ها:
- **PNG:** برای شفافیت (پیشنهادی)
- **SVG:** برای کیفیت بالا
- **JPG:** برای عکس‌های معمولی

### بهترین اندازه‌ها:
- **24×24 پیکسل:** استاندارد
- **32×32 پیکسل:** بزرگ‌تر
- **16×16 پیکسل:** کوچک

### نکات طراحی:
- رنگ آیکون سفید یا روشن باشد
- پس‌زمینه شفاف (PNG)
- حجم کم (زیر 50KB)
- طرح ساده و قابل تشخیص

